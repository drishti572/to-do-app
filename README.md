📝 Elegant To-Do List Web App

A modern and interactive To-Do List application built using HTML, CSS, and JavaScript.
It helps users manage daily tasks efficiently with a clean UI and powerful features.

🚀 Features
✅ Task Management
Add new tasks
Mark tasks as completed
Delete tasks
Clear all tasks
📅 Due Date & Reminder
Set due date and time
Get reminder alerts before deadline
🔄 Drag & Drop
Reorder tasks easily using drag & drop
🎯 Filters
View:
All tasks
Completed tasks
Pending tasks
🎬 Auto-Changing Background
Dynamic background images
Changes every few seconds
Smooth transitions
💾 Local Storage
Tasks are saved in browser
No data loss on refresh
🛠️ Technologies Used
HTML → Structure
CSS → Styling & animations
JavaScript → Logic & functionality
LocalStorage → Data persistence
📁 Project Structure
project/
│
├── index.html
├── style.css
├── script.js
▶️ How to Run
Download or clone the project
Open folder
Double click index.html

👉 That’s it! No installation required

💡 How It Works
Tasks are stored using LocalStorage
Background images change using JavaScript interval
Drag & drop uses HTML5 Drag API
Reminder uses setTimeout()
